httpx.vim plugin
====================

# Getting started

* Install httpx.vim manually: 

```
git clone https://github.com/httpx-sh/httpx.vim.git \
  ~/.config/nvim/pack/github/start/httpx.vim
```

* Open index.http file and move the cursor to a request, then invoke `:Httpx` command
